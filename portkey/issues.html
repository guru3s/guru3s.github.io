What users are stuck on

  • Provider correctness across Anthropic, Bedrock, Vertex, Groq, Hugging Face, Triton, Fireworks, and Cloudflare Workers AI.
  • Runtime behavior under streaming, header mutation, timeout, fallback, DNS, and response-normalization edge cases.
  • Configuration ergonomics: retry plus strategy, override parameters, importability, logging, and model endpoint discovery.

What people keep asking for

  • More native providers and model coverage, especially OCI, Requesty, AnyAPI, MiniMax, Databricks, Cerebras, OpenRouter Responses, and Kimi on Fireworks.
  • Stronger operational controls: batch APIs, metadata-based budgets, OTel metrics, better logs, and negative filters.
  • Gateway extensibility: Remote MCP, arbitrary passthrough providers, hook context metadata, and package imports.
